# combo-generator
Generates random hero combinations for dota 2.

## Sample input/output
run with `python run.py <int>`. The `int` parameter is defaulted to 1 if none is given

Input: `python run.py 3`  
Sample Output:
```

HERO 1:
---------------------------
Undying | Support
---------------------------


HERO 2:
---------------------------
Lone Druid | Roaming
---------------------------


HERO 3:
---------------------------
Rubick | Roaming
---------------------------

```
