# combo-generator
Generates random hero combinations for dota 2.

## Dependencies
- Python 3

## Running the combo-generator
run with `python run.py <int>`  
The `int` parameter is defaulted to 1 if none is given

## Sample input/output
Input: `python run.py 3`  
Sample Output:
```

HERO 1:
---------------------------
Undying | Support
---------------------------


HERO 2:
---------------------------
Lone Druid | Roaming
---------------------------


HERO 3:
---------------------------
Rubick | Roaming
---------------------------

```
